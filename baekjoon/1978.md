```java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.io.IOException;
import java.util.StringTokenizer;

public class Main {

    public static void main(String[] args) throws IOException {

        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int num = Integer.parseInt(br.readLine());

        StringTokenizer st = new StringTokenizer(br.readLine(), " ");

        int cnt = 0;
        int a;

        for (int i = 0; i < num; i++) {
            a = Integer.parseInt(st.nextToken());
            for (int j = 2; j <= a; j++) {
                if (j == a) {
                    cnt++;
                }
                if (a % j == 0) {
                    break;
                }
            }
        }
        System.out.println(cnt);
    }
}
```

소수는 1과 자기 자신만을 약수로 가지고, 1은 소수가 아니기 때문에 2부터 입력받은 수 직전까지 나머지를 검사한다.

**시간복잡도 : O(N)**


```java
public class Main {

    public static void main(String[] args) throws IOException {

        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        br.readLine();    
        int count = 0;

        StringTokenizer st = new StringTokenizer(br.readLine(), " ");

        while (st.hasMoreTokens()) {

            // 소수인경우 true, 아닌경우 false
            boolean isPrime = true;

            int num = Integer.parseInt(st.nextToken());

            if (num == 1) {
                continue;
            }
            
            for (int i = 2; i <= Math.sqrt(num); i++) {
                if (num % i == 0) {
                    isPrime = false;
                    break;
                }
            }
            if (isPrime) {
                count++;
            }
        }
        System.out.println(count);
    }
}
```

16의 약수는 다음과 같다.

- 1 X 16 = 16
- 2 X 8 = 16
- 4 X 4 = 16
- 8 X 2 = 16
- 16 X 1 = 16

4 X 4를 기준으로 양쪽이 대칭을 이루기 떄문에 소수인지 확인하기 위해서는 바로 가운데 약수까지만 나누어떨어지는 것을 확인하면 된다.
즉, **제곱근**까지만 확인하면 된다.

**시간복잡도 : O(√N)**

```java
boolean[] isPrime(int number) {

    boolean[] prime = new boolean[number + 1]; //0부터 시작하므로 +1

    prime[0] = true;
    prime[1] = true; //0과 1은 소수가 아니므로 true

    for (int i = 2; i <= Math.sqrt(number); i++) {
        if (prime[i] = true) { //이미 소수가 아닌 경우 다음 반복문으로 건너뛴다.
            continue;
        }

        for (int j = i * i; i < number + 1; j = j + i) {
            prime[j] = true;
        }
    }
    return prime; //배열의 인덱스가 소수라면 false
}
```

**시간복잡도 : O(nlog(logn))**
